## 🌙 about
this is a small interactive website created as a digital safe space  
for moments when someone misses me 💚  

it holds gentle words, comfort messages, and quiet reminders  
that you are never alone.

---

## 💌 how it works
- open the website link  
- enter the password: `missyou`  
- unlock a hidden message ✨  
- read whenever you need comfort 🌿  

---

## ✨ features
- 🌿 soft sage green aesthetic  
- 💌 password-protected letter  
- 💚 floating heart animations  
- 🌙 smooth transitions & cozy design
